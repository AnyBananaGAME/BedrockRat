## Installation

1. **Clone the BedrockRat repository:**

    ```bash
    git clone https://github.com/AnyBananaGAME/BedrockRat.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd BedrockRat
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

---

## Usage

To run BedrockRat, execute the following command:

```bash
node .
```
